Git upload project

						steps of upload a project:
1.$ git init
2.$ git add .
3.$ git commit -m "message"
4.$ git branch -M main
5.$ git remote add origin (url from github repository)
6.$ git push -u origin main


							Push changes


1.$ git add *
2.$ git commit -m "message"
3.$ git push -u origin main
